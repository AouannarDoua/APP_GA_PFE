The main objective of this DUT is to train specialized technicians in Web & Mobile application development, with an emphasis on:

  • Programming languages (C/C++, Java, PHP)
  • Information systems and databases
  • Computer architecture, systems, and networks
  • Designing, developing, and deploying Web and Mobile applications

It is also important to note that this program perfectly meets the qualification requirements. Furthermore, it is capable of effectively meeting the qualification needs of development projects installed in Nador. The aim is to train individuals with skills capable of providing scientific and fruitful support to these projects.
